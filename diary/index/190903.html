# 2019年9月3日

# 心烦的缘由

  每天总有一段时间心情是烦躁的，夏天更是如此。不知从什么时候开始，心里的烦恼越来越多，也许是因为“那件事”吧。我以前很内向，不太愿意与人交流，所以无人倾诉，即使有人愿意听，我也觉得说了也没用,所以心里的烦恼越积越多。后来脾气感觉变得不太好，我曾一度怀疑我有抑郁症，遇到不顺心的事情就发泄出来，把以前感觉受到的所有委屈的爆发出来，就好像上天再跟你做对一样，甚至还会跟朋友发脾气。因为这个原因，我失去了很多朋友，我很自责，我努力的让自己忍住，收住自己的脾气。慢慢的，我收起了脾气，但是治标不治本，忍不住的时候还是会发泄，每次特别烦躁的时候就觉得自己真没用，我这样想的原因大概是在这个烦恼的死循环里，我需要一个出口，一个借口，让自己退出这个死循环，让自己找个理由放弃，“我真是没用”，随着心里一直默念这几个字，我也变的不再挣扎，也不再烦恼。与其说不再烦恼不如说脑子里什么都没想，一片空白，什么也不想去想，什么也不想去做，大概，我又进入了另一个死循环吧。

  就那样，我一直压抑着自己，努力让自己不发脾气，过了很久很久。

# 人生远不止眼前的苟且

  我还是对我朋友发脾气了，我卖力地解释着，说自己已经很努力的在改变了，很努力了。“每个人都有这种时候，人家一年来一次，可是你呢？”。我不在说话，我又进入了死循环，我又开始找借口让自己退出死循环。“是我没用，对不起”。我又再一次失去了朋友。我开始反思，开始有打算，有规划，去感受生活的美好，我想变好，不想再让朋友担心。

  我下定了决心，这次要好好的改变，不是硬压着自己的脾气，是去感受生活的美好，即使有种种困难，种种压力，压迫着你，但这也是生活，是好多人想要的生活。我是幸运的，我应该感到很幸福，相比那些每天都在与病魔作斗争的人来讲，我这些都太卑微了，我还为这些小事情每天绞尽脑汁，我应该花更多的时间去作自己该作的事情，自己想做的事情。“好想能好好的工作一次，恋爱一次，旅行一次，孝顺一次。如果有有下辈子，我希望身体健康”，文章里的那个女孩的这句话深深的刻在了我的心里，也时时刻刻的提醒着我，不要因为一点小事而烦恼。

# 后记

  今天下班回家，跟一个老朋友分享了我写的一个关于垃圾分类的小程序，她夸了我，我很开心。她跟我分享她在学校里面的事情，我跟她讲了我开始写日记的事，她说她能出现在我的日记里了，我自己嘴角微微上扬，笑了起来。
