# 2019年9月2日

# 工作内容

# 1.导入数据生成预警工单并增加预警分数

前端文件通过axios将文件（excel）对象上传到后台解析并且做一系列验证过后，调用之前写到创建工单到方法生成工单，生成之后通过spring到监听组件执行增加预警分值的方法

# 遇到的问题

这些功能之前都已经接触过了，大同小异，所以做起来并没有遇到什么太难的点，主要花的时间就是各种参数的校验以及前端的校验，写完了在多测一测几乎就没什么问题了

# 总结

我很少写总结，但是最近遇到了很多事情，尤其是今天在坐地铁的时候“简书”看到了一篇文章，让我陷入了思考。一位患了白血病的女孩，开头就讲述了她这三年的经历。“今天注册了账号，因为三年来有太多太多的话憋在心里，怕自己有一天突然走了就说不出来了。”文章大概讲述了女孩三年来与病魔的斗争，以及她内心的一些想法，她只是想活着，想跟我们正常人一样活着，一次工作，一次恋爱，一次旅行，一次孝顺.....

文章写了很多，但这是冰山一角，数不尽的痛苦我们不曾体会，这个女孩一定还有好多话吧

文章的最后还不忘鼓励我们，让我们珍惜健康和生命，不要为了一些小事苦恼忧郁。是啊，还有舍么事比的上生命呢，我更加意识到人总有一死，你不知道哪天就会突然降临悲剧，我应该努力了，我不想给自己的人生留下什么遗憾
