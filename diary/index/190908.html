# 2019年9月8日

# 我的日常

  好几天没有写工作总结和日记了，这周也蛮烦的，工作上的事情又多了起来。周三晚上还跟两个学弟一起加了班，那天晚上老板又跟我讲周四又要出差一趟，要是以前我听到出差我内心是拒绝的，因为一想到出去要演示我写的系统，我就很慌，莫名的很慌，可能是怕出问题吧。但是那天晚上我内心竟然毫无波动（甚至有点想吃黄焖鸡米饭.jpg），大概是加班已经加到没有感觉了吧（主要是没吃饭加班）。周三加班快要结束的时候，我一个学弟突然说到“今晚不回去了吧，就睡公司了，回去也太晚了”。说完他就向刘佳成（老板的弟弟）问公司门的密码，并说明了原因，但是刘佳成并没有给，只是说让他问老板，学弟没有问，也没说话了。加完班，我收拾好东西准备走了，看到了学弟也收拾东西，问到他不是说睡公司么，他说不睡了，并且感觉他有点失落。我们都是顺路的，每天都会一起坐地铁，那天晚上下班他走的很快，并没有等我跟另一个学弟，我大概猜到可能是有点不开心了（用四川话讲就是冲气）。第二天早上一大早，刘佳成就给我发消息，给我讲了公司门锁密码，并让我给我那个学弟讲。我还在想为什么会这样，大概也是刘佳成想到了昨晚上没有给我学弟讲密码到事情可能让他有点不开心了。其实学弟问刘佳成密码到时候，刘佳成心里肯定也是担心到，毕竟公司那么多重要东西，学习心里想多可能就是我们都认识挺久了，也挺熟悉了，问密码应该也没什么问题的。这也不是一件什么大事，就很小的一件事，以后该怎么过还是怎么过，不会因为这件事而有所改变，我分析了这么多，也许是我想多了吧，我就是爱多想，做一件事总是要犹豫很久，总是过度考虑别人的感受，然后自己一直犹豫。我也知道犹豫就会败北，但是我总觉得不考虑周全更没有把握，我也不知道该怎么做，希望有一天我能够找到自己吧。。

  周四就是出差了，中午去的。那天下午是罗江区开的一个很重要的会议，很多领导，很多人，因为要演示系统，内心还是有一点紧张的。我和老板坐在最后面，听完了正常会议（我还是听的蛮认真的），大概就是讲怎么怎么扶贫呀，之前存在哪些问题，以后该怎么解决之类的。要轮到演示系统了，我还是我开始紧张起来......

  那天出差演示系统的时候确实很蛮紧张的，但是过了就好。经历了这么多，并没有什么迈不过去的坎，这一点我还是在我写程序的时候最先领悟到。写程序经常会遇到一些问题，甚至当时我觉得这永远都解决不了，但最后都是能顺利的完成的，时间一长，我再次遇到困难时也不会慌张，因为我知道我一定会解决的，人生亦是如此。

  周末也过的很快，我没做什么就已经过去了...今天早上6点过我就被地震摇醒了，当时感觉摇得不厉害，加上困意，我甚至都没站起来，一直躺着，还在想如果摇得更厉害了我要不要跑，我住这么高跑不下去怎么办...果然我还是想的很多，犹豫就会败北。今天我还是忍不住找了她，有很多话想讲，但是又怕......我不能再这样想了，我还是要用行动证明。

  明天周一，开始工作，加油！
