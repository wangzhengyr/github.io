# 2019年9月2日

# 新的起点

  以前我总是自己反思，老是立下flag，却老是做不到，典型的想的比做的多，但是经历了种种事情，我想我应该行动起来，光想是不行的。曾经老想着想开始写日记，记录每天发生的事情，每天自己心里想到事情，哪怕是每天只写一点点。

# 生命真的很脆弱

  今天的雨好大，有点凉。每天上下班的地铁上，我都会翻出手机，看一些文章。今天看到一篇文章，标题吸引了我“你不知道会有多坚强，因为除了坚强，你别无选择”，文章的开头就讲了这个女孩跟病魔斗争了三年。“今天注册了账号，因为有三年有太多的话憋在心里”，这是她开头讲述的一段文字，后来看到这里还打多了一个“有”字，当我看完了她的故事，我在看到这里多打的字，心中就会想起她当时在病床上的情景，我想哭，我很害怕，我不知道是对病魔的恐惧还是同情这个女孩。

  我继续往下面读，看着这些文字，我脑海里浮现出了她讲述得画面，一个活泼得女孩一点一点的被病魔所折磨。“从前最怕打针的我生活开始与点滴为伴，抽血成了每天的日常，可怕又必须要做的骨穿与腰穿也数不清，头发一次次掉光，想吃的不能吃，到哪里口罩不离，生活中必须处处注意。好想能好好的工作一次，恋爱一次，旅行一次，孝顺一次。如果有下辈子，我希望身体健康”。看到这里，我身体经开始颤抖，手中的雨伞好像已经快要拿不住，鼻子酸酸的，眼睛里已经有点湿润了。

  我不敢在继续往下看，关上了手机，眼睛闭起来。

  我缓了缓，我准备关注这个女孩好好的看她的文章，想看着她慢慢的好起来。但我关注完想看她的别的文章的时候，我心一颤，感觉这一刻我好像没有了呼吸。一共只有四篇文章，前两篇是17年1月份写的，第三篇是4月份，文章标题是转发女儿的遗书。我点了进去，看着一张张女孩写的遗书，眼中的泪水终于忍不住往下掉了出来，我又关上了手机，我不敢再看下去，我很害怕，我不知道是对失去生命的畏惧还是对女孩的惋惜

# 请珍惜每一天

  我关上了手机，脑海里浮现出太多画面，又一片空白。我不知道自己在想什么，不知道每天都在做什么，想要什么，常常因为一些很小的事情而发愁，忧郁，而且还不知道为什么。

  之后我还是忍不住，继续的看了下去。这些文字一个一个的刻在我的心里。

  “一天晚上，被痛的实在受不了，难以语言描述，妈妈正好进来拿药给我吃，结果我控制不住地朝她发脾气了。大家一直叫我要听话要坚强，可我也是人啊 不是钢铁 坚强久了也会累。我终于扛不住了 我已经被病痛折磨的神智不清了，我需要发泄。其实我不坚强，我也害怕，我大哭了一场，哭的很痛快很痛快，爸妈看我这么难受也看不下去了，大家都憋不住了，撕心裂肺的哭了，怪上天，怪它为什么对我这么残忍？”

  文章的最后女孩还不忘鼓励我们，可能那时她正在忍受着疼痛。是啊，生活中即使在多不愉快的事情，在多烦心的事情，在生命和健康面前都太卑微了，如果把每天都当作人生中最后一天活着，相信会很多很多想要做的事情吧。女孩最后说的一番话就是想让我们好好活着，不要因为一点小事苦恼忧郁，觉得生活没意思。

  人生的路虽然很长，但不要总以为时间还很多，就今天要做的事情往后面推，因为你永远不知道明天和意外哪一个先来。珍惜每一天吧，没准别人正在努力的与时间作斗争，与生命赛跑，而且我却在肆意挥霍。

  晚上下班，依旧下着雨，很冷。回到家，我又回想起今天发生的事情，我应该记录下来，我不能只光想了，我要行动起来。
